# Tiny Sports — Brand Specification

Source of truth for fonts and colours, extracted from this repo (`src/app/layout.tsx`, `src/app/globals.css`). Use for all documents and print collateral.

## Fonts
- Headings: **Plus Jakarta Sans** (Google Fonts, free)
- Body: **Inter** (Google Fonts, free)

## Colours
| Role | Hex | CMYK |
|---|---|---|
| Primary (teal) | `#005F4A` | 100 / 0 / 23 / 63 |
| Teal mid (links, fills) | `#007A5F` | 100 / 0 / 22 / 52 |
| Teal bright (highlights) | `#009476` | 100 / 0 / 20 / 42 |
| Accent (amber) | `#FC9F30` | 0 / 37 / 81 / 1 |
| Body ink | `#0F1216` | near-black |
| Muted text | `#5D646F` | — |
| Pale tint (panels) | `#EBF4F1` | — |
| Status open / closed | `#3A9742` / `#C53637` | — |

## Print application rule
- **White page background** on every page.
- **Body text in dark ink** (`#0F1216`).
- **Accent colours (teal, amber) are for logos, headers/header bars and borders/rules only.** No accent or dark page/panel floods. The website is a dark-background design; print is the opposite.
